

## 📡 مستندات کامل API

**Base URL:** `http://127.0.0.1:8000/api/`

### ۱. کاربران (Users)

| متد     | آدرس                  | توضیح                     | دسترسی        |
|---------|-----------------------|---------------------------|----------------|
| `GET`    | `/users/`             | لیست همه کاربران          | ادمین          |
| `POST`   | `/users/`             | ایجاد کاربر جدید          | همه            |
| `GET`    | `/users/{id}/`        | جزئیات یک کاربر           | ادمین          |
| `PUT`    | `/users/{id}/`        | ویرایش کاربر              | صاحب کاربر/ادمین |
| `DELETE` | `/users/{id}/`        | حذف کاربر                 | ادمین          |

**مثال ایجاد کاربر:**
```bash
curl -X POST http://127.0.0.1:8000/api/users/ \
  -H "Content-Type: application/json" \
  -d '{
    "name": "علی احمدی",
    "username": "ali_ahmadi",
    "password": "secret123"
  }'
```

### ۲. دسته‌بندی‌ها (Categories)

| متد     | آدرس                     | توضیح                     |
|---------|--------------------------|---------------------------|
| `GET`    | `/categories/`           | لیست همه دسته‌بندی‌ها     |
| `POST`   | `/categories/`           | افزودن دسته‌بندی جدید     |
| `GET`    | `/categories/1/`         | جزئیات یک دسته‌بندی       |
| `PUT`    | `/categories/1/`         | ویرایش دسته‌بندی          |
| `DELETE` | `/categories/1/`         | حذف دسته‌بندی              |

**مثال افزودن دسته‌بندی:**
```json
POST /api/categories/
{
  "farsi": "پیتزا",
  "english": "Pizza"
}
```

---

### ۳. آیتم‌های منو (Menu Items)  🍕

| متد     | آدرس                          | توضیح                              |
|---------|-------------------------------|------------------------------------|
| `GET`    | `/menu/`                      | لیست همه غذاها                     |
| `GET`    | `/menu/?category=1`           | فیلتر بر اساس دسته‌بندی             |
| `GET`    | `/menu/?search=پپرونی`        | جستجو در نام و توضیحات             |
| `POST`   | `/menu/`                      | افزودن غذای جدید                   |
| `GET`    | `/menu/5/`                    | جزئیات یک غذا                     |
| `PUT`    | `/menu/5/`                    | ویرایش غذا                        |
| `DELETE` | `/menu/5/`                    | حذف غذا                           |

**مثال افزودن پیتزا پپرونی:**
```bash
curl -X POST http://127.0.0.1:8000/api/menu/ \
  -H "Content-Type: application/json" \
  -d '{
    "name": "پیتزا پپرونی ویژه",
    "description": "پپرونی مرغوب، پنیر موزارلا فراوان، سس مخصوص",
    "price": 298000,
    "category": 1
  }'
```

**پاسخ موفق (۲۰۱ Created):**
```json
{
  "id": 12,
  "name": "پیتزا پپرونی ویژه",
  "description": "پپرونی مرغوب، پنیر موزارلا فراوان، سس مخصوص",
  "price": "298000.00",
  "category": {
    "id": 1,
    "farsi": "پیتزا",
    "english": "Pizza"
  }
}
```

**فیلترهای موجود در `/menu/`:**
- `?category=1` → فقط پیتزاها
- `?search=مارگاریتا` → جستجو در نام و توضیحات
- `?ordering=-price` → گران‌ترین اول
- `?ordering=price` → ارزان‌ترین اول
